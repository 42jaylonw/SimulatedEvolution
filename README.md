## Quickstart
1. Install python tutorial [here](https://kinsta.com/knowledgebase/install-python/).
2. Install python dependencies
    ```commandline
    pip install -r requirments.txt
    ```
3. run sim
   ```commandline
   python random_moving.py
   ```
   press `q` to quit the simulation
